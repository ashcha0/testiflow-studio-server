# 城市媒资库

这个目录用于存储城市相关的视频素材，供视频生成系统使用。

## 添加视频素材

请在此目录下添加城市相关的视频素材，例如：

- 摩天大楼.mp4
- 繁华街道.mp4
- 城市夜景.mp4
- 交通系统.mp4
- 城市公园.mp4
- 文化场所.mp4
- 智能科技.mp4

## 视频素材要求

1. 视频格式支持：MP4、AVI、MOV、MKV
2. 建议分辨率：1920x1080（全高清）
3. 建议时长：5-30秒
4. 文件名将作为关键词用于检索，请使用有意义的名称

## 注意事项

- 视频文件名应与脚本中的关键词相匹配，以便系统能够正确检索
- 可以创建子目录进一步组织视频素材，如：城市/夜景/、城市/建筑/等
- 子目录名也将作为关键词用于检索
